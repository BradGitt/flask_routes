Flask Routes

This Flask app shows some simple examples of routes in Flask as well as how to differentiate between different HTTP Methods. e.g. GET and POST.

You may try the following routes and see what the output is:

  • Default About Page - The route is /about
  • Learn about Tom - The route is /about/Tom
  • Add 5 + 11 - The route is /add/5/11. Only integers allowed.
  • Add 2.3 + 7.7 - The route is /add-floats/2.3/7.7 Only floats allowed.
  • View a static blog post - The route is /post/1
  • Return a static file - The route is /robots.txt
